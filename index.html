A Collaborative Project for University of Michigan's Introduction to Applied Data Science (SI568) Class

By Lauren Asam and Michael Mitchell

Introduction

Google Vision API is a powerful tool that allows you to analyze images and extract information from them. In this project, we learned how to use the Google Vision API to analyze images and extract information from them. By the end of this project, we gained a good understanding of how to use the Google Vision API and how to integrate it into our own projects.
